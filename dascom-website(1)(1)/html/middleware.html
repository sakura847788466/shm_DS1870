中间件云打印解决方案
Cloud Print Middleware Solution
得实云打印中间件（Middleware）是处于用户操作系统和打印机之间的应用软件，
起到中间枢纽，交换信息的作用。
什么是中间件 ？
得实云打印中间件是基于云服务开发的系统集成组件，
具有独立部署、运行及使用特点。
我们在本地，他们在云端
不用为收费、数据、控制等问题担忧
技术框架
使用前沿流行的微服务架构及Docker容器
自主研发
从打印机、通讯模块、通讯协议到后台都是自主研发
服务升级
基于多年打印机SDK、插件、技术支持等开发经验打造的系统集成方案
面向企业
有一定开发能力或特殊功能需求的企业用户
系统架构
我们的优势
专业性强
自主研发通讯协议，专业性强。业务模块化、专业化、可根据用户要求快速开发定制及扩展。
可控性高
使用高通WIFI芯片自主搭建硬件平台，可控性高。
可靠性高
基于TCP/IP协议开发，可靠性高。
可定制化需求
支持多种业务需求及软件定制要求。
硬件延展性强
自主研发的硬件平台可支持串口、USB口等硬件接口扩展。支持扫描枪、RFID、电子称等多种设备整合方案实现。
数据安全性高
不同于第三方云平台，得实打印中间件所有服务均部署在客户的企业服务器上，数据安全及可控性高。
